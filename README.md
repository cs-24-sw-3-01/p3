# KMD Vacationplanner

## Running SvelteKit
First navigate into the `frontend` folder. Then run the following:
```Bash
bun run dev -- --open`
```

## Running Spring Boot
First navigate into the `backend` folder. Then run the following:
### MacOS
```Bash
./gradlew bootRun
```
### Windows
```Bash
.\gradlew.bat bootRun
```
